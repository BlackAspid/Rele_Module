# Rele Module

_This is a rele module for electronic projects. It´s can used with &copy;Arduino, &copy;Microchip PIC.... Design in [KIcad](https://www.kicad.org/)._

## Starting 🚀

_You can clone this repo with this command, first go to the desired folder in your terminal._

```git clone https://github.com/BlackAspid/Rele_Module.git```

### Pre-requirements 📋

* Install [Kicad](https://www.kicad.org/download/) Vanilla from this link.

### Setting up 🔧
_Clone this repository_

```
git clone https://github.com/BlackAspid/Rele_Module.git
```

_Open Kicad program_

![Kicad Logo](/IMG/kicadlogo.png)

_Press File -> Open project (Ctrl + O)_

![Kicad Interface](/IMG/kicadinterface.png)

_Select file .pro_

![Kicad file .pro](/IMG/kicadfile.png)

_Here we go that's all, you can add or remove things_
